# Detect memory leaks in a given code snippet

## In-depth Explanation (Nvidia Interview Style)

- Memory leaks occur when allocated memory is not freed.
- Use tools like `valgrind`, `asan`, or custom wrappers to detect leaks.
- In interviews, explain how to instrument code and interpret leak reports.

### Interview Tips
- Be ready to discuss static vs dynamic analysis, and how leaks impact long-running systems.
