# Pointer arithmetic problems

## In-depth Explanation (Nvidia Interview Style)

- Pointer arithmetic is fundamental in C for array and structure manipulation.
- Understand how pointer increments depend on the type size.
- Be able to explain pointer difference, pointer to pointer, and pointer casting.

### Interview Tips
- Be ready to write code that manipulates arrays and structures using pointers.
- Know how pointer arithmetic works with different data types and memory alignment.
