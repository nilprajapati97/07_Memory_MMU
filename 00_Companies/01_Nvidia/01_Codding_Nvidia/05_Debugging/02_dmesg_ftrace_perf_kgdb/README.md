# Q47: Use dmesg, ftrace, perf, kgdb

## In-depth Explanation (Nvidia Interview Style)

- `dmesg`: Kernel log buffer for runtime messages.
- `ftrace`: Function tracer for profiling and debugging.
- `perf`: Performance analysis tool for profiling CPU, memory, and events.
- `kgdb`: Kernel debugger for live debugging.

### Interview Tips
- Be ready to discuss use cases and configuration for each tool.
