# Directory Navigation Guide

This guide helps you quickly find topics and questions layer by layer.

## Top-Level Layers

1. **Core_Linux_Kernel_C**
   - Fundamental C and kernel idioms
2. **Synchronization**
   - Locks, concurrency, and race conditions
3. **Memory_Management**
   - Allocation, paging, and memory safety
4. **Device_Drivers**
   - Character devices, IOCTL, polling
5. **Debugging_Questions**
   - Kernel debugging, tracing, bug detection
6. **NVIDIA_GPU_Relevant_Topics**
   - GPU, DMA, PCI, cache coherency
7. **NVIDIA_Priority**
   - High-yield, must-know interview topics
8. **NVIDIA_Realistic_Questions**
   - Realistic, multi-layered interview problems

---

## Example Navigation

- To study memory barriers: `NVIDIA_Priority/10_memory_barriers/`
- To review device driver polling: `Device_Drivers/30_poll_select_epoll_support/`
- To debug use-after-free: `Debugging_Questions/49_debug_use_after_free/`
