# Bit Manipulation Macros

## Approaches

### Approach 1: Basic Macros
- Standard bit operations
- Side-effect safe with parentheses
- Interview standard

### Approach 2: Advanced Macros
- Multi-bit operations
- Mask creation
- Field extraction

### Approach 3: Inline Functions (C99+)
- Type-safe
- Debuggable
- Modern C approach

## Macro Best Practices
- Always use parentheses
- Avoid side effects
- Use `U` suffix for unsigned
- Test with complex expressions
